# Projeto-War-C

Projeto acadêmico desenvolvido para a faculdade utilizando a linguagem C, com foco em lógica de programação e conceitos fundamentais de desenvolvimento de software.

## 📚 Sobre o projeto

O Projeto-War-C foi criado como atividade acadêmica para praticar conceitos importantes da programação em C, incluindo:

- Estruturas condicionais
- Laços de repetição
- Funções
- Manipulação de variáveis
- Organização de código
- Lógica de programação

---

## 💻 Tecnologias utilizadas

- Linguagem C
- GCC Compiler
- Git & GitHub

---

## 🎯 Objetivo do projeto

Aplicar na prática os conhecimentos aprendidos durante o curso de Engenharia de Software, desenvolvendo raciocínio lógico e boas práticas de programação.
